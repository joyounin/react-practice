ex03: webpack 개발서버

1. 설치 패키지
[ex01]$ npm i -D webpack webpack-cil webpack-dev-server

2. 번들링 환경(webpack.config.js) 설정
    - devServer

3. 스크립팅
    "scripts": {
        "start": "npx webpack serve --progress"
        "build": "npx webpack"
    }

3. 빌드
[ex01]$ npm run build

4. 실행
[ex01]$ npm start
